Newton's Second Law with Angles

Newton’s Second Law can be used to solve systems involving forces at angles. The strategy:

  • Draw a force diagram
  • Label all known forces
  • Calculate gravity and any components of angled forces
  • Sum forces in the x-direction: if there is acceleration, \[\Sigma F_x = ma_x\]
  • Sum forces in the y-direction: if there is acceleration, \[\Sigma F_y = ma_y\]
Example 1:
A student pushes a 15.0 kg box with a 50 N force directed 30° below the horizontal. The box accelerates at 3.0 m/s². What is the normal force on the box? What is the friction force on the box?
Show/Hide Answer


Step 1: Resolve the force into components
\[ F_x = 50\cos 30^\circ = 43.3\,\text{N} \]
\[ F_y = 50\sin 30^\circ = 25\,\text{N} \]
\[ W = mg = 15 \times 9.8 = 147\,\text{N} \]
Step 2: Find the normal force (vertical direction)
The vertical forces cancel (no vertical acceleration):
\[ \Sigma F_y = 0 \] \[ N + (-25) + (-147) = 0 \] \[ N = 172\,\text{N} \]
Step 3: Find the friction force (horizontal direction)
The box accelerates horizontally:
\[ \Sigma F_x = ma \] \[ f + F_x = ma \] \[ f + 43.3 = 15 \times 3.0 \] \[ f + 43.3 = 45 \] \[ f = 1.7\,\text{N (to the left)} \]
The normal force is 172 N. The friction force is 1.7 N to the left.

Example 2:
A student applies a 250 N force to a 70 kg box at an angle 60° above the horizontal. The box experiences 60 N of friction. What is the normal force on the box? What is the acceleration of the box?
Show/Hide Answer


Step 1: Resolve forces
F_g = mg = 70 \times 10 = -700\,\text{N} F_g = mg = 70 \times 10 = -700\,\text{N}
F_{x} = 250\cos 60^\circ = 125\,\text{N} F_{x} = 250\cos 60^\circ = 125\,\text{N}
F_{y} = 250\sin 60^\circ = 216.5\,\text{N} F_{y} = 250\sin 60^\circ = 216.5\,\text{N}

Step 2: Find the normal force (vertical direction)
No vertical acceleration:
\Sigma F_y = 0 \Sigma F_y = 0 F_g + F_y + N = 0 F_g + F_y + N = 0 -700 + 216.5 + N = 0 -700 + 216.5 + N = 0 N = 483.5\,\text{N} N = 483.5\,\text{N}
Step 3: Find the acceleration (horizontal direction)
\Sigma F_x = ma \Sigma F_x = ma -60 + 125 = 70a -60 + 125 = 70a 65 = 70a 65 = 70a a = 0.93\,\text{m/s}^2 a = 0.93\,\text{m/s}^2
The normal force is 483.5 N. The acceleration is 0.93 m/s² to the right.
