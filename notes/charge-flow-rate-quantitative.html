Understanding Current

‚ö° What is Current?

  • Current (I) is the flow rate of charge through a circuit
  • Measured in amperes (amps) = coulombs/second
  • Consistent with arrow directions drawn on circuits
  • Measured using ammeters wired in series

üìä Current in Series Circuits

Key Principle: Current is identical everywhere in a single loop

Ibattery = IA = IB = IC

Note: Higher resistance results in lower overall current

üìä Current in Parallel Circuits

Key Principle: Current splits at branches, sum equals total

Ibattery = IA + IB + IC

Note: Higher resistance in a branch means less current through that branch

Circuit Examples

Series Circuit Example

View Solution

Given: Current through battery wire = 0.5 A

Solution:

In a series circuit, current is the same everywhere:

Ibattery = IA = IB = IC = 0.5 A

This is true regardless of individual bulb resistances!

Parallel Circuit Example

View Solution

Given: Battery current = 0.6 A

Case 1: Equal resistances

  • Current splits equally
  • IA = IB = IC = 0.2 A

Case 2: Bulb C has double resistance

  • IA = IB = 0.24 A
  • IC = 0.12 A (half due to double resistance)

Mixed Circuit Example

View Solution

Given: Current through bulb A = 0.5 A

Case 1: Equal resistances

  • Second branch has double resistance (two bulbs)
  • Isecond branch = 0.25 A
  • Ibattery = 0.75 A

Case 2: Bulb C has double resistance

  • Second branch has triple resistance
  • Isecond branch = 0.17 A
  • Ibattery = 0.67 A
